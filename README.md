# OmniDocs

A unified Python toolkit for document processing models.

## Installation

```bash
pip install omnidocs
```

Or for development:

```bash
git clone https://github.com/adithya-s-k/OmniDocs.git
cd OmniDocs
uv sync
```

## Quick Start

```python
import omnidocs

print(omnidocs.__version__)
```

## Development

Run tests:

```bash
uv run pytest
```

## Requirements

- Python 3.10 - 3.11
